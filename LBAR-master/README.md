# LBAR
Location-Based Augmented Reality

-Name Layar 13 "radar"
-Open SampleScene
-Enter your desired lat,lon on POI game object
-Add LBARCam component to Wikitude Camera object
-(on iOS) give camera permission to your app
-Build and run
